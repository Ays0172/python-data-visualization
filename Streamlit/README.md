# 📊 Interactive Data Visualization Learning App

A comprehensive, interactive Streamlit application that teaches Python data visualization step-by-step using **Matplotlib**, **Seaborn**, and **Plotly**.

## 🚀 Getting Started

```bash
# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```

## 📚 What's Covered

| # | Page | Topics |
|---|------|--------|
| 1 | Getting Started | First chart, library imports |
| 2 | Line Charts | Basic, multi-line, time series |
| 3 | Bar Charts | Grouped, stacked, histograms |
| 4 | Scatter Plots | Color/size encoding, bubble, regression |
| 5 | Statistical Plots | Box, violin, stem plots |
| 6 | Area & Fill | Area charts, stack plots, fill between |
| 7 | Pie & Donut | Pie, donut, exploded, shadows |
| 8 | Categorical Plots | Count, strip, swarm, point plots |
| 9 | Joint & Pair Plots | Hue/style/size, heatmaps |
| 10 | Subplots & Layouts | Grid layouts, shared axes |
| 11 | Styling & Aesthetics | Colors, themes, fonts, scales |
| 12 | 3D Plots | Scatter3D, surface, interactive |
| 13 | Advanced | Live data, datetime, export |

## 🎮 Features

- **Step-by-step learning** — click buttons to reveal each concept
- **Library switcher** — render the same chart with Matplotlib, Seaborn, or Plotly
- **Code display** — see the actual Python code for every visualization
- **Custom datasets** — upload your own CSV or use built-in sample data
